Welcome to SpatialInformation documentation
===========================================

**SpatialInformation** is a Python library intended to facilitate the analysis of calcium imaging data in the context of virtual reality experiments and spatial cognition.
It implements a variety of algorithms developed by the field of spatial cognition over the years and can be integrated into an easy pipeline to run on your data. 

.. note::

   This project is under active development.
